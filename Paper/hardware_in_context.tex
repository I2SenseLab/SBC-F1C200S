\section{Hardware in context}
% Include a short description of the hardware, putting into context of similar open hardware and proprietary equipment in the field.
The Simple SBC board is a reasonable replacement for a lot of Raspberry Pi computers used in Research applications. 
The board is not intended to be a direct substitute, 
but instead an alternate design path when a full Raspberry Pi is more powerful than a system requires or not customizable enough for the application. 
This work is based on building out a design based on guidance from the Linux Business Card \cite{hilliard_business_card_linux_2019} and 
So you want to build an embedded Linux System blog posts \cite{carlson_embedded_linux}.
