\textbf{Abstract}\\ 
The Raspberry Pi Computer is an incredible tool for numerous research applications. 
However, despite wide usage it is not ideal for all research applications. 
Using a Raspberry Pi can increase costs, overall prototype size, and decrease flexibility. 
For this reason we have designed a simple single board computer based around a low-cost MPU available for cheap assembly. 
The design pairs the F1C200S Allwinner MPU with the RP2040 MCU. 
The operating system can be loaded either onto the attached microSD slot or loadded directly onto the on-board flash.
The design is open source and can be used either as is with a large area dedicated to support a daughter board for specialized 
research applciations, or the design can be customized directly to integrate within another project. 
This makes the system ideal for long term data-logging in environments where using a Raspberry Pi would represent a significant unneeded cost.
